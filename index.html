서버 소개

근명고등학교 LABS는 개발 실습, 프로젝트 협업, 디지털 자료 관리, 모니터링 학습 등 다양한 교육 활동을 지원하기 위해 구축되었습니다. 학생 중심 운영을 지향하며, 체계적인 접근 제어와 자동화된 백업으로 안정적인 학습 인프라를 제공합니다.

제공 서비스

소스 코드 저장소

GitLab 기반의 사설 저장소로 팀 프로젝트 관리 및 CI 파이프라인 실습 가능.

학습 플랫폼

Moodle를 활용한 과제 제출, 토론, 온라인 퀴즈 제공.

파일 협업

Nextcloud로 문서 · 이미지 · 연구 자료 안전한 공유.

모니터링 대시보드

Grafana & Prometheus로 서버 상태 및 리소스 시각화.

자동화 / CI

Jenkins / GitLab CI를 통한 테스트 · 빌드 자동화 학습.

컨테이너 실습

Docker / Podman 환경에서 서비스 배포 및 운영 경험.

주요 특징

  • 안정성: RAID 스토리지 + 정기 백업으로 데이터 보호
  • 보안: 인증·권한 분리, HTTPS, 접근 로그 감시
  • 확장성: 컨테이너 기반 서비스 확장 구조
  • 교육용 설계: 실제 DevOps 흐름 체험 가능
  • 모니터링: 실시간 리소스/서비스 상태 학습 자료화
  • 커뮤니티: IT 동아리 중심 운영·기여 프로세스

하드웨어 / 인프라 사양

서버 하드웨어

  • CPU: AMD Ryzen 7 (8C/16T)
  • RAM: 32GB DDR4
  • Storage: NVMe 1TB + HDD 4TB (RAID1)
  • NIC: 2.5GbE

소프트웨어 스택

  • OS: Ubuntu Server 22.04 LTS
  • Container: Docker / Compose
  • DB: PostgreSQL / MariaDB
  • Reverse Proxy: Nginx

운영 / 관리

  • 백업: 증분 + 주간 전체
  • 모니터링: Prometheus + Grafana
  • CI: GitLab Runner / Jenkins
  • 이슈 추적: GitLab Issues

보안

  • 방화벽: UFW 정책 제한
  • 접근: SSH 키 + MFA(순차 도입)
  • HTTPS: 자동 인증서 갱신
  • 로그: 중앙화 & 감사

* 실제 사양은 시점에 따라 조정될 수 있습니다.
